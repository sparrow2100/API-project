Women Composers of Classical Music API

Endpoints

Endpoint URL Method Successful Response Error Response Request Parameters
Create a new user /users POST Status: 201
Body: A JSON object with data about the user, and a unique ID. Ex: { "name": "Jane Doe", "favoriteComposers: []", "id": "123456", } 		Status: 400
Body: "user needs a name" 		Body: A JSON object with data about the user. Ex: { "name": "Jane Doe", "favoriteComposers: []", }
Add a composer to favourite composers /users/:id/:composerName POST Status: 200
Body: A message, ex. "Elizabeth Maconchy has been added to user 123456's favorite composers" 		Status: 400
Body: "user not found" 		Path parameter: The name of the composer to be added.
Delete a favourite composer /users/:id/:composerName DELETE Status: 200
Body: ex. "Hildegard von Bingen has been removed from user 123456's favorite composers" 		Status: 400
Body: "user not found" 		Name of the composer to be deleted
Delete user /users/:id DELETE Status: 200
Body: "user 123456 has been deleted" 		Status: 400
Body: "user not found" 		id of user to be deleted
Update the name of a user /users/:id PUT Status: 200
Body: A JSON object with the user's information, including the updated name.
Ex: { "name": "updatedName", "id": "123456", } 		Status: 400
Body: "user not found" 		The new user name
Get all composer data /composers GET Status: 200
Body: A JSON object with all the data about women composers 		None None
Get data about a composer using her name /composers/:name GET Status: 200
Body: A JSON object with data about the composer
Ex. (not the full amount of data) { "name": "Elizabeth Maconchy", "life": { "fullName": "Elizabeth Maconchy", "lifespan": "1907-1994", "bio": "placeholder", "nationality": "Irish/English" },} 		Status: 400
Body: "composer not found" 		The name of the composer
Get name and description of era /composers/era/:eraName GET Status: 200
Body: A JSON object with information about the era
Ex: { "name": "baroque", "description": "The baroque era, flourishing from the early 17th to the mid-18th century, was marked by ornate and expressive music, defined by elaborate ornamentation and the emergence of instrumental forms like the concerto and the sonata. " } 		Status: 400
Body: "no such era" 		The era name
Get life info about composer based on full name /composers/life/:fullName GET Status: 200
Body: JSON object with information about a composer's life.
Ex. { "fullName": "Elizabeth Maconchy", "lifespan": "1907-1994", "bio": "placeholder", "nationality": "Irish/English" } 		Status: 400
Body: "life does not exist" 		The composer's full name
