Women Composers of Classical Music API

Endpoints

Endpoint URL Method Successful Response Error Response Request Parameters
Create a new user /users POST Status: 201
Body: A JSON object with data about the user, and a unique ID. Ex: { "name": "Jane Doe", "username": "exampleusername", "password": "examplepassword", "email": "name@email.com", "birthday": "09-09-1999", "favoriteComposers: []", "id": "123456", } 		Status: 400
Body: "Error: (error)" 		Body: A JSON object with data about the user. Ex: { "name": "Jane Doe", "username": "exampleusername", "password": "examplepassword", "email": "name@email.com", "birthday": "09-09-1999", "favoriteComposers: []", "id": "123456", }
Add a composer to favourite composers /users/:username/favouriteComposers/:composerId POST Status: 201
Body: json with the user information, including the ID for the new composer added to the array of favouriteComposers. 		Status: 500
Body: "Error: (error)" 		Path parameter: The ID of the composer to be added, and the username of the user.
Delete a favourite composer /users/:username/favouriteComposers/:composerId DELETE Status: 201
Body: json with the user data, excluding the ID of the composer that was just deleted from the favouriteComposers array. 		Status: 500
Body: "Error: (error)" 		ID of the composer to be deleted, and the username of the user.
Delete user /users/:username DELETE Status: 200
Body: "username was deleted" 		Status: 400
Body: "user not found" 		username of user to be deleted
Update user info by username /users/:username PUT Status: 200
Body: A JSON object with the user's information, including the updated info.
Status: 500
Body: "Error: (error)" 		The new user info, username
Get all composer data /composers GET Status: 201
Body: A JSON object with all the data about women composers 		Status: 500
Body: "Error: (error)" 		None
Get data about a composer using her name /composers/:name GET Status: 200
Body: A JSON object with data about the composer
Status: 404
Body: "composer not found" 		The name of the composer
Get name and description of era /eras/:name GET Status: 200
Body: A JSON object with information about the era
Ex: { "name": "baroque", "description": "The baroque era, flourishing from the early 17th to the mid-18th century, was marked by ornate and expressive music, defined by elaborate ornamentation and the emergence of instrumental forms like the concerto and the sonata. " } 		Status: 404
Body: "Era not found." 		The era name
Get info about all eras /eras GET Status: 200
Body: JSON object with information about all eras. 		Status: 500
Body: "Error: (error)" 		None
